2.5 EXPORT OF CREYOCO MODULES

The following two steps are required to export learning modules:
1) Choosing a layout
2) Export of the learning module

1) Choosing a layout

Click on the link “Choose layout” in the menu. There are two different layout options available. Each layout on the right represents a responsive website, i.e. the website adapts optically to the size of the respective terminal screen.

Click on the desired layout. This will open a preview of the page. You can close the preview window by clicking on the cross icon on the top right.

Note: Not all features are displayed correctly in the preview. Therefore, you should check your learning modules later in the exported version.

2) Export of the learning module

Click on the link “Download package” in the menu. Creyoco offers different export options: IMS Package, Common Cartridge, SCORM 2004, SCORM 1.2 and website (zip).

The learning module is exported with the layout that you selected in the last step.

Export as web page

If you want to publish your learning module directly on the Internet in the form of a website, choose the export option “website (zip)”. In this case creyoco produces an HTML-based Web page with a front page “index.html” with your learning module. If you copy the entire contents of the folder and upload them on a web server, you can launch the finished website immediately.
