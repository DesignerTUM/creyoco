1 LEARNING MODULES AND MODULE EDITORS

A multimedia-based learning module is a didactically structured tutorial. The content can be displayed through texts, hyperlinks, pictures, graphics, data files for download, audio, video etc. You can create interactive education modules by integrating applets and exercises (multiple choice questions, cloze tests, reflective tasks). Simple creation of modules is possible with the module editor.

What is a learning module editor?

„Editor tools serve for the creation of multimedia-based learning sequences and presentations with a programming knowledge as little as possible and facilitate the creation of e-learning media for a wide group of users.“

(Loos/Zimmermann/Chikova 2008)

Characteristics of the education module editor

Simple, intuitive operation without programming knowledge
Authors without any knowledge of programming language or scripting language can easily create a learning module.

Creation of structured learning content
The available functionalities serve the didactical preparation of the course content.

Modular design
The author of the education module can choose between a variety of tools (text processing, integration of pictures etc.) and also combine them.

Unrelated learning modules
The learning module is not enclosed to a platform and therefore both usable as website and can be embedded into a platform.

(Vgl. Nieweler 2006, Roche 2007)
