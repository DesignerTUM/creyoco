1.1 DEVELOPMENT OF LEARNING MODULES AND WEBSITES

Creyoco is a learning module editor for arranging different types of teaching and learning content (text, pictures, data for download, audio and video material etc.) as autonomous module. Besides the simple creation of learning modules and websites, interactive elements like exercises for self-testing purpose can be embedded into the content.

Learning modules and websites are created easily in a short time and automatically get navigation placed on the left side. The contents of modules can be used with different layouts in different contexts. Afterwards the learning modules can be used in various output formats: The learning module can be offered on a website or be embedded in a learning platform and made available thereby.

The SCORM standard is able to send the page views and the e-test results to a learning platform, if needed; the users data can be tracked from there.

Creyoco can be used for free on www.creyoco.de.

If necessary, the software can be downloaded and run on a private server (open source).

Creyoco: General Characteristics

  • Separation of layout and content by „styles“
  • Hierarchical structure: Unlimited number of layers
  • Didactical tools like previous knowledge
  • Interactive learning elements: Test questions (multiple choice, cloze tests), reflection
  • Workshop instructions, password-protected text
  • Text editor with integration of media

Background information on Creyoco

  • Developed in the Media Center of the TUM
  • Advancement of the editor „eXe-Learning“
  • Web-based and independent of the operating system
