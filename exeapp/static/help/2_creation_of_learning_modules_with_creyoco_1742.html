2. CREATION OF LEARNING MODULES WITH CREYOCO

To create a learning module (also a website) with creyoco the following basic steps are recommended:

1)   Create new learning module

A new learning module is created in the course overview.

2)   Creating a structure (navigation) for the learning module

You can add pages, remove pages, rename pages and duplicate pages. The sites can be moved up or down and the level of detail of the pages can be changed in the navigation.

3)   Pasting content using the instructional Devices (iDevices)

Texts, web sites, tests etc. can be inserted in pre-fabricated structures by iDevices. The iDevices are structured into topics.

4)   Choosing a layout

There are two different layouts to choose from. For each layout a version with responsive web design is available.

5)   Export of the learning module

For the publication as a web page or in a learning platform the learning module can be exported as website (Zip), IMS Package, Common Cartridge, SCORM 2004 and SCORM 1.2.
